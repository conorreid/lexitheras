# Example Usage

## Search by title or author:
python lexitheras.py iliad                    # Searches for texts containing "iliad"
python lexitheras.py homer                    # Searches for all texts by Homer
python lexitheras.py symposium                # Finds Plato's and Xenophon's Symposium
python lexitheras.py anabasis                 # Searches for Anabasis

## Direct URN (if you know it):
python lexitheras.py urn:cts:greekLit:tlg0012.tlg001.perseus-grc2

## List all available texts:
python lexitheras.py --list-texts

## Search only (don't create deck):
python lexitheras.py plato --search-only

## Custom output and deck names:
python lexitheras.py iliad -o my_iliad_deck.apkg -n "Homer's Iliad Vocabulary"
